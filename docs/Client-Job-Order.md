## 2. Scope of work (summary)

The Provider has performed (or will complete, as per your agreed timeline) **custom design and full development** of the Client’s **online shop website**, including:

- Visual design (layout, typography, colors, responsive behavior) aligned with the Client’s brand  
- All agreed **web pages** and sections (home, shop, product pages, categories, devotionals, journaling / club hub, our story, contact, checkout, shipping, privacy, and related templates)  
- **Shopping cart** and **checkout flow** with order summary suitable for **Messenger / m.me** (or agreed channel)  
- **Content management (CMS)** so the Client can update products, homepage content, navigation, and site settings without code  
- **Contact** page and form / email forwarding setup as configured  
- **Live chat** (free-tier provider, e.g. Tawk.to) wired in settings  
- **Deployment** on **free static hosting** (e.g. GitHub Pages) with automated build/deploy (e.g. GitHub Actions)  
- Short **handover** documentation or walkthrough for routine CMS updates  

**Operating model (Client benefit):** Hosting, chat, and forms are chosen to use **free or low recurring cost** tiers where possible, so monthly platform fees stay minimal. That does not reduce the value of the **one-time design and build** covered by this Job Order.
